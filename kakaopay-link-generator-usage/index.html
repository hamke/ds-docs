카카오페이 모바일 송금 링크 자동 생성 서비스 사용 방법

업데이트 : --01

디소스(Desource)가 카카오페이 모바일 송금 링크 자동 생성 서비스 사용 방법을 소개해 드립니다.

◼︎ 사용 방법

카카오페이 모바일 송금 링크 자동 생성 서비스 메인 페이지에서 “QR 코드 이미지 파일 선택” 버튼을 클릭해서, 본인의 카카오페이 QR 코드(수신용) 이미지 파일을 선택합니다.

서비스를 이용하기 전, 카카오페이 QR 코드 이미지가 준비되어 있지 않다면, 먼저 본인의 카카오페이 QR 코드 이미지 파일을 다운로드합니다. (필수)

내 카카오페이 QR코드(수신용) 확인 및 다운로드 방법 : 바로 가기

QR 코드 이미지 파일을 선택한 뒤, “업로드 및 작업 진행” 버튼을 클릭합니다.
“잠시만 기다려 주세요” 메시지가 나오며, 카카오페이 모바일 링크 생성 작업이 시작됩니다.

카카오페이 모바일 송금 링크 자동 생성 작업이 시작되면서 결제창이 나옵니다.
결제창 내의 QR 코드를 카카오톡으로 스캔하거나 카카오톡으로 결제 정보 메시지를 송신하여, 결제를 완료합니다.

결제를 완료하면, 업로드된 카카오페이 QR 코드에 대한 모바일 링크가 곧바로 화면에 표시됩니다.
생성된 모바일 링크를 복사해서, 홈페이지, 블로그, 또는 SNS(인스타그램, 페이스북, 트위터 등)에서 자유롭게 사용할 수 있습니다.
